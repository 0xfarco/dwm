# DWM

dwm (dynamic window manager) is an extremely fast, small, and dynamic window manager for X.

## Patches

- restartsig 
- restore after restart 
- vanitygaps 
- hide vacant tabs 
- pertag 
- move stack
- attach bottom
- actual full screen
- always center
- systray

## Installation

```bash
git clone https://github.com/0xfarco/dwm.git
cd dwm
sudo make clean install
```
